# canvas（请点右上角star）
小程序canvas 2微码 生成图片并保存到手机相册

# 概述
最近学习小程序canvas的使用，在各种文档的帮助下，写了一个demo。
并使用分享产生一直含有2微码的图片。
如果你需要网络图片 请使用wx.downloadfile
